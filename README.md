# Adtek Labs — Zoiper QR Generator

A modern, web-based utility designed to quickly generate QR codes for provisioning SIP accounts on the Zoiper softphone app.

🌍 **Live Demo:** [https://adtek93.github.io/zoiper-qrcode/](https://adtek93.github.io/zoiper-qrcode/)

## 🚀 Features

*   **Modern Glassmorphism Design:** A beautiful, responsive user interface with an animated gradient background.
*   **Dark & Light Mode:** Toggle between themes effortlessly with automatic system preference detection.
*   **Seamless Integration:** Fully compatible with Zoiper OEM API for one-time QR code provisioning.
*   **Flexible Security:** Passwords and credentials are automatically URL-encoded, allowing the use of complex passwords without arbitrary restrictions.
*   **Zero Storage:** Everything runs client-side. We DO NOT store or transmit your SIP credentials to any third-party servers other than the official Zoiper API.
*   **Clean Architecture:** Built with HTML, Vanilla CSS, and JavaScript.
